# Accelerometer

The architecture of this device: 

➢ Implemented I2C communication between the MPU6050 and the ESP32-S3.

➢ Used SPI protocol to run the Nokia5110 LCD for data visualization.

➢ Processed raw sensor data to display acceleration and angular velocity values.

➢ Designed and fabricated a custom PCB voltage regulator to provide a steady 5V supply.
